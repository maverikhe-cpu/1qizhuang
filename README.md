# 一起装网站 (YiQiZhuang Website)

This project is a React-based website for "一起装", a smart delivery and digital marketing solution for decoration companies.

## Tech Stack

- **Framework**: Vite + React
- **Routing**: React Router DOM
- **Styling**: Vanilla CSS (CSS Variables + Flexbox/Grid)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Project Structure

- `src/components`: Reusable components (Navbar, Footer) and page-specific components (home/*).
- `src/pages`: Main page components (Home, etc.).
- `src/styles`: Global styles and variables.

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Design Implementation

The design follows the requirements document:
- **Colors**: Primary Blue (#1A3250), Accent Orange (#FF7A00).
- **Typography**: Montserrat/Source Han Sans for headings, Inter for body.
- **Animations**: Smooth transitions, hover effects, and data visualizations.
